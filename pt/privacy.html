Política de Privacidade

Última atualização: 9 de dezembro de 2025

O Mindful Breaker ("nós", "nosso" ou "nos") está comprometido em proteger sua privacidade. Esta Política de Privacidade explica como nosso aplicativo opera em relação aos seus dados pessoais.

1. Sem coleta de dados (Apenas local)

Acreditamos que seu bem-estar digital é um assunto privado. O Mindful Breaker não coleta, armazena ou transmite quaisquer dados pessoais para servidores externos.

Todos os dados gerados pelo aplicativo (estatísticas de uso, histórico, configurações, códigos PIN) são armazenados localmente em seu dispositivo usando o armazenamento interno seguro do Android. Se você desinstalar o aplicativo, esses dados serão excluídos permanentemente.

2. Inventário de aplicativos instalados

Para fornecer a funcionalidade principal de bloquear distrações, o aplicativo precisa verificar e listar os aplicativos instalados em seu dispositivo.

Como usamos isso: Acessamos sua lista de aplicativos instalados exclusivamente para exibi-los a você, permitindo que você selecione quais aplicativos deseja bloquear ou limitar. Esta lista é processada localmente em seu dispositivo e nunca é enviada para nenhum servidor.

3. Divulgação da API de serviços de acessibilidade

Importante: O Mindful Breaker usa a API de Serviços de Acessibilidade do Android (Accessibility Service API) para funcionar. Esta é a permissão principal necessária para o funcionamento do aplicativo.

Por que usamos isso?

  • Para detectar qual aplicativo está sendo executado atualmente em primeiro plano.
  • Para verificar se o aplicativo em execução está na sua "Lista de bloqueados".
  • Para exibir uma tela de bloqueio ou uma sobreposição de atenção plena quando você excede seu limite.

O que NÃO fazemos:

  • NÃO lemos seus e-mails, mensagens ou verificamos suas senhas.
  • NÃO alteramos suas configurações sem sua permissão.
  • NÃO rastreamos sua digitação (keylogging).
  • NÃO enviamos nenhum conteúdo da tela para nenhum servidor.

4. Privacidade das crianças (Modo Criança)

Nosso aplicativo inclui um recurso "Modo Criança" projetado para pais. Este modo permite definir um código PIN para impedir a alteração de limites.

O aplicativo não coleta informações pessoais de crianças. O código PIN é armazenado localmente no dispositivo como um hash seguro (criptografado) e nunca é transmitido.

5. Otimização da bateria

Para funcionar corretamente, o aplicativo pode solicitar ignorar as otimizações de bateria. Isso é necessário para garantir que o serviço de bloqueio não seja encerrado pelo sistema Android enquanto você estiver usando outros aplicativos.

6. Alterações nesta Política de Privacidade

Podemos atualizar nossa Política de Privacidade de tempos em tempos. Portanto, aconselhamos que você revise esta página periodicamente para quaisquer alterações. Notificaremos você sobre quaisquer alterações publicando a nova Política de Privacidade nesta página.
