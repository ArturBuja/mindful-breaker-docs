Política de Privacidad

Última actualización: 9 de diciembre de 2025

Mindful Breaker ("nosotros", "nuestro" o "nos") se compromete a proteger su privacidad. Esta Política de Privacidad explica cómo opera nuestra aplicación con respecto a sus datos personales.

1. Sin recopilación de datos (Solo local)

Creemos que su bienestar digital es un asunto privado. Mindful Breaker no recopila, almacena ni transmite ningún dato personal a servidores externos.

Todos los datos generados por la aplicación (estadísticas de uso, historial, configuraciones, códigos PIN) se almacenan localmente en su dispositivo utilizando el almacenamiento interno seguro de Android. Si desinstala la aplicación, estos datos se eliminan permanentemente.

2. Inventario de aplicaciones instaladas

Para proporcionar la funcionalidad principal de bloquear distracciones, la aplicación necesita escanear y listar las aplicaciones instaladas en su dispositivo.

Cómo lo usamos: Accedemos a su lista de aplicaciones instaladas únicamente para mostrárselas, permitiéndole seleccionar qué aplicaciones desea bloquear o limitar. Esta lista se procesa localmente en su dispositivo y nunca se sube a ningún servidor.

3. Divulgación de la API de servicios de accesibilidad

Importante: Mindful Breaker utiliza la API de servicios de accesibilidad de Android (Accessibility Service API) para funcionar. Este es el permiso principal requerido para que la aplicación funcione.

¿Por qué lo usamos?

  • Para detectar qué aplicación se está ejecutando actualmente en primer plano.
  • Para verificar si la aplicación en ejecución está en su "Lista de bloqueados".
  • Para mostrar una pantalla de bloqueo o una superposición de atención plena cuando excede su límite.

Lo que NO hacemos:

  • NO leemos sus correos electrónicos, mensajes ni verificamos sus contraseñas.
  • NO cambiamos su configuración sin su permiso.
  • NO rastreamos su escritura (keylogging).
  • NO enviamos ningún contenido de pantalla a ningún servidor.

4. Privacidad de los niños (Modo Niños)

Nuestra aplicación incluye una función "Modo Niños" diseñada para padres. Este modo permite configurar un código PIN para evitar cambios en los límites.

La aplicación no recopila información personal de niños. El código PIN se almacena localmente en el dispositivo como un hash seguro (encriptado) y nunca se transmite.

5. Optimización de la batería

Para funcionar correctamente, la aplicación puede solicitar ignorar las optimizaciones de batería. Esto es necesario para garantizar que el servicio de bloqueo no sea eliminado por el sistema Android mientras usa otras aplicaciones.

6. Cambios en esta Política de Privacidad

Podemos actualizar nuestra Política de Privacidad de vez en cuando. Por lo tanto, se le aconseja revisar esta página periódicamente para ver si hay cambios. Le notificaremos de cualquier cambio publicando la nueva Política de Privacidad en esta página.
